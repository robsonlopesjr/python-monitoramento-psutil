## ▶️ Como execultar a aplicação

### Criar ambiente virtual

```
python -m venv venv
```

### Ativar ambiente virtual

```
.\venv\Scripts\activate
```

### Instalar os pacotes do arquivo requirements
```
pip install -r requirements.txt
```
